# task-manager

##

#### web

Clone o repositório:
```bash
git clone https://github.com/brunomat480/task-manager.git
```

Baixe as dependencias:
```bash
npm install
```
```bash
composer install
```
##

#### web
Banco de dados:
```bash
php artisan migrate
```
```bash
php artisan db:seed
```

##

#### web
inicie a aplicação:
```bash
npm run dev
```
abra outro terminal e execute o comando:
```bash
php artisan dev
```
